How to use

◆ご利用方法

  1. チャイムを押さずに玄関ドアからお入りください。靴を脱いでお上がりいただき、スタッフがご案内するまでお待ちください（スリッパの用意もございます）。
  2. 中の引き戸を猫が出ないように開け、入られたらすぐに締めるようお願いします（万一ねこが玄関スペースに出てしまったら、すぐにスタッフにお声をかけて教えてください）
  3. そのまま階段を上がり、スタッフの案内に従って手洗い・消毒をしてください。
  4. 空いているお好きなお席におかけいただき、ドリンク類やスイーツをご注文ください。ご注文時に時計を見ながら利用開始時間をスタッフと確認しましょう。
  5. 猫たちを眺めたり遊んだりして満足いくまでのんびりとお過ごしください。終了時刻の確認は特にしませんので、こちらからお声かけは致しません。
  6. 終了される場合はスタッフにお声をおかけいただき、レジにてお支払いください。
